<h3>Features</h3> 
<hr>

- **Home Page**: Introduction and brief overview.
- **About Page**: Detailed information about me, my skills, and experience.
- **Projects Page**: Gallery of my projects with descriptions and links.
- **Contact Page**: Form to get in touch with me.

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**

## How to Use

1. Clone the repository:
    ```bash
    git clone https://github.com/prajjukorban/my-potrfolio.git
    ```
2. Navigate to the project directory:
    ```bash
    cd my-potrfolio
    ```
3. Open the `index.html` file in your preferred web browser:
    ```bash
    open index.html
    ```

## Project Structure

- `index.html` - Home page.
- `about.html` - About page.
- `projects.html` - Projects page.
- `contact.html` - Contact page.
- `style.css` - CSS file for styling.
- `script.js` - JavaScript file for interactivity.
